# Pearl Spices Barcode Generator

A Python-based barcode generation tool for creating **POS-friendly, easy-to-scan barcodes** for Pearl Spices products.

The barcodes are optimized with:
- Wider bars
- Adequate quiet zones
- Reduced height for compact spice labels
- High DPI for professional printing
